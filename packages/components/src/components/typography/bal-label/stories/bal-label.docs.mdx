## Examples

<span id="story--components-typography-label--required-and-optional" style={{ opacity: 0 }}></span>

### Required and Optional

If the `required` is set to `false` the optional suffix will be added. The suffix is available for all
the supported languages.

<Canvas>
  <Story id="components-typography-label--required-and-optional" />
</Canvas>

<span id="story--components-typography-label--states" style={{ opacity: 0 }}></span>

### States

Two states are supported for the label component to match the form control states.

<Canvas>
  <Story id="components-typography-label--states" />
</Canvas>

<span id="story--components-typography-label--sizes" style={{ opacity: 0 }}></span>

### Sizes

The size `small` is the default for form control labels.

For advanced form controls like `bal-radio-button` or `bal-checkbox-button` we use the other two sizes.
