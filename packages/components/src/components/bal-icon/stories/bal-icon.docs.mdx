<bal-doc-lead>
  <b>Icons</b> are pure svg files. To add a new icon just place your svg file into the{' '}
  <a href="https://github.com/baloise/design-system/tree/master/packages/icons/svg">folder</a>. Then the build script
  will automatically optimize the svg and provide it to the component. It is important that the svg icons are made with
  fill elements instead of strokes.
</bal-doc-lead>

There are three ways in which the Baloise Design System offers support for icons:

1. Standardized [Design System icons](?path=/docs/foundation-icons-overview--page#overview).
2. Custom SVG icons [passed into the component](?path=/docs/components-icon--basic#svg).
3. Custom SVG added to the [Design System config](?path=/docs/foundation-icons-development--page#import-during-initialization) during the initalization phase.

> **TIP**
>
> The bal-icon is based on the [icons foundation](?path=/docs/foundation-icons-development--page) so check this out for adding more icons to your set.
