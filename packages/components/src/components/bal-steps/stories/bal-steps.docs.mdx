The step option can be passed in with the property too. For that we provide a util
function `newBalStepOption`.

```typescript
import { newBalStepOption } form '@baloise/design-system-components'

const options = [
  newBalStepOption({ label: "My Label 1", value: "1" }),
  newBalStepOption({ label: "My Label 2", value: "2" }),
]

// next pass the option array to the component ...
```
