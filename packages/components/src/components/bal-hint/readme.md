### bal-hint
 
#### Properties

| Property     | Attribute     | Description                                      | Type      | Default   |
| ------------ | ------------- | ------------------------------------------------ | --------- | --------- |
| `closeLabel` | `close-label` | Text for the close button.                       | `string`  | `'Close'` |
| `small`      | `small`       | Disables the close button for tablet and desktop | `boolean` | `false`   |


#### Methods

##### `close() => Promise<void>`

Closes the hint box.

###### Returns

Type: `Promise<void>`



##### `open() => Promise<void>`

Opens the hint box.

###### Returns

Type: `Promise<void>`



##### `toggle() => Promise<void>`

Toggles the hint box.

###### Returns

Type: `Promise<void>`
