# Datepicker

A datepicker allows manual date entry as well as open the dropdown content with a calendar to select a date.

<Canvas className="has-visible-overflow">
  <Story id="components-form-datepicker--basic" />
</Canvas>

## Manual Input

To enable manual input remove the property `readonly` and add the property `trigger-icon`.
The `trigger-icon` property only opens the dropdown when the icon gets clicked.

## Usage

To set the date use the [date utils](https://github.com/baloise/web-app-utils/blob/master/packages/utils/README.md).

```typescript
import { newDateString, now } from '@baloise/web-app-utils'

const datepickerDisabled = document.getElementById('datepicker-disabled')
datepickerDisabled.value = newDateString(now())
// or
datepickerDisabled.value = newDateString(2020, 0, 13)
// or
datepickerDisabled.value = newDateString(new Date(2020, 0, 13))
```
