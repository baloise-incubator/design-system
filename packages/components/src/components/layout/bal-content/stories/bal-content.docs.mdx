### Alignment

The children of the content can be aligned `start` (default), `center` or `end`.

<Canvas>
  <Story id="components-layout-content--alignment" />
</Canvas>

<span id="story--components-layout-content--direction" style={{ opacity: 0 }}></span>

### Direction

Default direction is vertically, but can be changed by setting the property `direction` to `row`.

After that the content is rendered horizontally.

  <Story id="components-layout-content--direction" />
</Canvas>
