<bal-doc-lead>
  The <b>Stack</b> element is responsible for arranging its direct descendants in a vertical or horizontal direction,
  while also allowing for the inclusion of optional spacing between each child.
</bal-doc-lead>

> The **Stack** component deals with layouts in one dimension, whereas the [Grid](?path=/docs/foundation-grid-development--breakpoints-css-utility) manages layouts in two dimensions.
> By default, the **Stack** component stacks children vertically in a column.

### Direction

The default arrangement for Stack is a vertical column. Nevertheless, the direction prop allows items to be positioned horizontally in a row as well.

  <Story id="components-layout-stack--direction" />

### Alignment

With the direction `column` the children of the stack can be aligned to `start` (default), `center` or `end`.
