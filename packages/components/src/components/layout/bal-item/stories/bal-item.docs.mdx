## Examples

<span id="story--components-layout-item--direction" style={{ opacity: 0 }}></span>

### Direction

Default direction is vertically, but can be changed by setting the property `direction` to `row`.

After that the content is rendered horizontally.

<Canvas>
  <Story id="components-layout-item--direction" />
</Canvas>

<span id="story--components-layout-item--alignment" style={{ opacity: 0 }}></span>

### Alignment

With the direction `column` the children of the item can be aligned to `start` (default), `center` or `end`.

  <Story id="components-layout-item--alignment" />
</Canvas>
