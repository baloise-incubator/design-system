# Support

<bal-doc-lead>Support is provided by the Baloise Design System team for users of the system.</bal-doc-lead>

> Before contacting us take some time to explore the contents of this documentation.

## Contributing

Please have a look at the [contributing guidelines](?path=/docs/development-contributing-overview--page).

## Create an Issue

All you need is a [public GitHub account](https://github.com/) to get started. Most contributions begin with a GitHub issue using one of these templates:
