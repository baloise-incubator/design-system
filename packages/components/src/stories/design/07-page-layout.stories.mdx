# Page Layout

This is the page layout we recommand to use. The `bal-app` class enables the Baloise Design System styles underneath it.

```html
<body>
  <bal-app has-sticky-footer>
    <main class="container">
      <!-- Page content -->
      <bal-heading>Hello World<bal-heading>
    </main>
    <footer class="footer">
      <!-- Footer content -->
    </footer>
  </bal-app>
</body>
```

- The `.container` is a simple utility class that allows you to center content on larger viewports and to reduce the main space.
  - Use `container is-compact` for calculator application to reduce the max-width to 768px.
  - [Bulma Documentation to the container](https://bulma.io/documentation/layout/container/)
- `has-sticky-footer` attribute sticks the footer element to the bottom of the page.
