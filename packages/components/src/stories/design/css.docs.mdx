There are nine responsive mixins, which can be easily used in your sass files.

Use the scss mixins in your application the following:

```css
@import 'node_modules/@baloise/design-system-next-components/src/styles/global.utilities';

.bal-element {
  width: 100%;
}

@include tablet() {
  .bal-element {
    width: 380px;
  }
}

@include desktop() {
  .bal-element {
    width: 570px;
  }
}
```

The file `global.utilities` provides all defined scss variables like [colors](?path=/docs/design-colors--page) and also the mixins.

<h2 id="container">Container</h2>

The usage of the container is documented on the [Page Layout Page](?path=/docs/design-page-layout--page).

| container       | css                        | usage                                                       | max-width |
| --------------- | -------------------------- | ----------------------------------------------------------- | --------- |
| **default**     | `container`                | Default container for the most use cases.                   | 992px     |
| **detail-page** | `container is-detail-page` | Special use case for the website.                           | 744px     |
| **compact**     | `container is-compact`     | Should be used for sales funnels and small forms.           | 896px     |
| **blog-page**   | `container is-blog-page`   | Special use case for the website.                           | 984px     |
| **wide**        | `container is-wide`        | Should be used to table based applications.                 | 984px     |
| **fluid**       | `container is-fluid`       | For large application to use the whole width of the screen. | 100%      |

Building a columns layout with Bulma is very simple:

2. Add as many column elements as you want

<h2 id="spacing">Spacing</h2>

The CSS Framework provides **margin** `m*` and **padding** `p*` helpers for all directions:

- `*t` for **top**
- `*r` for **right**
- `*b` for **bottom**
- `*l` for **left**
- `*x` **horizontally** for both left and right
- `*y` **vertically** for both top and bottom

You need to combine a **margin/padding** prefix with a direction suffix. For example:

- for a `margin-top`, use `mt-*`
- for a `padding-bottom`, use `pb-_`
- for both margin-left and margin-right, use `mx-_`

For **flexbox** there are spacing values also available as `gaps`.

- for a `gap`, use `fg-*`

The **values (\*)** are documented in the token section

- [Spacing Tokens](?path=/docs/design-tokens--page#spacing)

<h2 id="flexbox">Flexbox</h2>

Helpers for all Flexbox properties

Combined with `is-flex`, all of the Flexbox CSS properties are available as CSS helpers:

- flex-direction
- flex-wrap
- justify-content
- align-content
- align-items
- align-self
- flex-grow
- flex-shrink

> These helpers are based on the [Bulma CSS Framework](https://bulma.io/documentation/helpers/flexbox-helpers/).

### Usage

This section has some examples how to use the provided flexbox css class helpers.

#### Centered

To center a child in a parent element use the following classes:

- `is-flex` to enable display flex
- `is-justify-content-center` to set the child horizontally to center
- `is-align-items-center` to set the child vertically to center

# Installation

The CSS Framework is provided as a [NPM package](https://www.npmjs.com/package/@baloise/design-system-next-css).

> **TIP**
> The CSS Framework package is included in the `@baloise/design-system-next-components` package and also in the proxy libraries.

To install the CSS Framework directly to your project, run the following command:

```bash
npm install @baloise/design-system-next-css
```

After the installation include the provided CSS or Sass file into your project.

Here is a basic HTML5 example:

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta http-equiv="X-UA-Compatible" content="IE=edge" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>CSS Framework - Example</title>
    <link rel="stylesheet" href="node_modules/@baloise/design-system-next-css/css/baloise-design-system.css" />
  </head>
  <body class="bal-app has-sticky-footer has-background-grey-0">
    <main class="container py-4">
      <section class="has-background-white has-radius has-shadow p-4 mb-4">
        <h1 class="title is-3">Hello World</h1>
      </section>
    </main>
  </body>
</html>
```
